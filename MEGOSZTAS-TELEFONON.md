# Matek Muhely megosztasa telefonra

## Javasolt megoldas

Tedd fel a `matek-gyakorlo` mappa tartalmat egy statikus webtarhelyre, peldaul GitHub Pagesre vagy Netlifyra. A diakok ezutan egy linket vagy QR-kodot kapnak.

## Diakoknak

1. Nyisd meg a linket telefonon.
2. Ird be a neved.
3. Gyakorolj a temakorokkel.
4. A haladas ezen a telefonon megmarad.
5. Idonkent nyomd meg a `Haladas mentese` gombot.
6. Tanev elejen mutasd meg az appot vagy kuldd el a mentett haladasfajlt.

## Tanarnak

- A `Haladas mentese` egy kodolt `.mmh` haladasfajlt tolt le.
- A fajl visszatoltheto a `Haladas betoltese` gombbal.
- A kituntetesek ikonra kattintva latszik, hany feladatot oldott meg a tanulo az adott temakorben.
- Ha a diak bongeszo-adatokat torol vagy masik telefont hasznal, a helyi haladas elveszhet, ezert fontos az idonkenti mentes.

## Feltolteskor

A teljes mappa tartalma kell:

- `index.html`
- `styles.css`
- `script.js`
- `manifest.webmanifest`
- `service-worker.js`
- az ikonfajlok

Feltoltes utan a kezdolap az `index.html` legyen.
